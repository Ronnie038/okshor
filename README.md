## Run Locally

Clone the project

```bash
  git clone https://github.com/Ronnie038/okshor.git
```

Go to the project directory

```bash
  cd okshor
```

Start the client

```bash
  cd client
```
Install dependencies

```bash
  npm install
```

Start the server

```bash
  cd server
```
Install dependencies

```bash
  npm install
```
```bash
  npm run dev
```
